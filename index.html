Welcome to the pymultinest tutorial!¶

This tutorial was developed for the Monte Carlo Methods in Advanced Statistics Applications and Data Analysis school (18-22 November 2013, MPI Munich). There are accompaning slides.

It should kickstart you to

  • implement your own likelihood and prior
  • run pymultinest
  • visualize and interpret the results
